# 🎮 English Sentence Game 2026

This is a fun English learning web game for students.  
Students must make a correct sentence using the given word.  
The game checks grammar, spelling rules, and gives scores automatically.

---

## 🌟 Features

- Random English verbs
- Timer for each question
- Smart sentence checker
- Score counter (out of 5)
- Chocolate reward system 🍫
- Auto next question
- Google Sheets score saving (optional)

---

## 🧑‍🎓 How to Play

1. Enter your name
2. Click **Start**
3. Type a proper sentence using the given word
4. Click **Check Answer**
5. Complete 5 questions to see your score

---

## 🏆 Rewards System

| Score | Reward |
|-------|---------|
| 5/5 | Kopiko Chocolate |
| 4/5 | Melody Chocolate |
| Below 4 | Good Try Message |
